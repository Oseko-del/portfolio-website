Building & Streamlining a Chatbot Website

One of the most exciting projects I worked on was integrating an AI-powered chatbot into a pharmacy website. The goal was simple help users get quick answers about prescriptions without needing to call or visit the pharmacy in person.

The Problem

Pharmacies receive dozens of repetitive questions daily. Things like drug interactions, dosage information, and prescription refills take up staff time that could be spent on more critical tasks. I wanted to automate those responses using AI.

The Solution

I integrated the Google Gemini API into a plain HTML/CSS/JS frontend with a Flask backend. The chatbot could understand natural language questions and respond with accurate, helpful information about medications and prescriptions.

Tech Stack Used

Frontend: Plain HTML, CSS, JavaScript — keeping it lightweight and fast with no heavy frameworks.

Backend: Python Flask — handling API calls to Gemini and serving responses back to the frontend.

AI: Google Gemini API (free tier) — powering the natural language understanding and responses.

What I Learned

This project taught me how to properly structure API requests, handle errors gracefully, and design a conversational UI that feels natural to use. It also showed me how powerful free AI APIs can be when used creatively in real world applications.

The result was a working prescription helper that reduced repetitive questions and gave users instant answers a real improvement in how the pharmacy interacted with its customers online.
